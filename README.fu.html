Introducing FoxUp
FoxUp is a mode-based markup language, putting emphasis on a "define-and-reset" markup style.

Mode-based?
FoxUp supports both an Input Mode and a Markup Mode.

By default, FoxUp runs in Input Mode. In this mode, every character is interpreted as pure text input and not processed, with the exception of the special character (*), used to switch to Markup Mode. To use this symbol as text in a document, simply enter it twice.

To switch to Markup Mode, simply type an * character in your document when in Input Mode. In Markup Mode, every key is interpreted as a command, making it easy to manage styles. For example, b activates bold, i puts text into italics and bi applies both. To switch back to Input Mode, simply press . (dot).

Commands
Note: lowercase characters affect a region until they're reverted. Other characters do not need to be closed, as they do not affect a region.

Character Name Description Parameters
* Markup Mode Switch to Markup Mode No
. Input Mode Switch to Input Mode No
b Bold Applies bold style No
s Size Changes the font size Required (higher = bigger)
i Italic Applies italic style No
R Revert Reverts style(s) Optional - Revert all by default. If given, revert last X styles
| Table Add table column No
To parse a document, simply enter the directory FoxUp is in and run python foxup.py <filename>. FoxUp will return the parsed document in the directory the original document was in.
